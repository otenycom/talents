# Approved Compact Telegram Format for All-In Portfolio Briefings

The abstract template for portfolio briefings generated from All-In episodes. The
worked example with the right tone/density is in the parent `SKILL.md`.

## Structure

**All-In | [Date]** (E? — [video_id])
*[Full Episode Title]*

**Core Thesis:** [one-line summary of the main investment theme]

### Key Takeaways
- Bullet 1 (claim + attribution + numbers when present)
- Bullet 2
- Bullet 3 (max ~5)

### Company Positioning
- **Company / TICKER** — [Bullish/Neutral/Bearish] | [compact one-line reason]
- ...

**Portfolio Watch:** [one actionable sentence]

## Rules

- Keep it compact and scannable for Telegram (under ~2k chars).
- Bold for company names and section headers.
- Avoid long paragraphs; focus on signals, positioning, and actionable implications.
- Only companies the hosts actually discussed; cite real numbers from the transcript.
- Do not add extra sections unless explicitly requested.
